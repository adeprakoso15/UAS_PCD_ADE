==============================================
DESKRIPSI PROGRAM
==============================================

Program ini merupakan aplikasi Pengolahan Citra
Digital berbasis GUI menggunakan Python.

Aplikasi dapat digunakan untuk:
- Menampilkan citra
- Konversi warna
- Peningkatan kualitas citra
- Transformasi geometri
- Menyimpan hasil pengolahan citra

==============================================
FITUR APLIKASI
==============================================

1. Load Citra
2. RGB ke Grayscale
3. RGB ke HSV
4. Brightness
5. Contrast
6. Histogram Equalization
7. Gamma Correction
8. Rotasi 90 Derajat
9. Flip Horizontal
10. Edge Detection
11. Simpan Hasil

==============================================
LIBRARY YANG DIGUNAKAN
==============================================

1. tkinter
   Digunakan untuk membuat GUI aplikasi

2. cv2 (OpenCV)
   Digunakan untuk pengolahan citra digital

3. Pillow (PIL)
   Digunakan untuk menampilkan gambar pada GUI

4. numpy
   Digunakan untuk operasi array dan gamma correction

==============================================
CARA MENJALANKAN PROGRAM
==============================================

1. Install Python atau Anaconda terlebih dahulu

2. Install library yang dibutuhkan:

   pip install opencv-python
   pip install pillow
   pip install numpy

3. Buka Anaconda Prompt

4. Masuk ke folder project:

   cd UTS_PCD_24041013_ADETEGUHPRAKOSO
   cd src

5. Jalankan program:

   python main.py
